# MCP Context7

Integration with Context7 for accessing official documentation and up-to-date technical references.

## Capabilities

- **Official Documentation**: Access AWS, Python, and framework documentation
- **API References**: Look up API signatures and usage examples
- **Best Practices**: Retrieve official best practices and guidelines
- **Version-Specific Info**: Get documentation for specific versions
- **Code Examples**: Find official code samples and patterns

- Lambda function configurations and limits
- DynamoDB API and access patterns

- CloudWatch Logs Insights query syntax

### 2. Python Library Documentation
- Pydantic v2 validators and field types

### 3. Framework Documentation

### 4. Best Practices
